## 📥 Tải xuống

Phiên bản mới nhất của **NVM-TOOL-PC** được phát hành tại GitHub Releases:

🔗 https://github.com/nvmaudio/NVM-TOOL-PC/releases/tag/NVM-TOOL-PC

### Các gói phát hành

- `Tool.DSP.PC_XX.zip` – Phiên bản Portable (không cần cài đặt)

### Hướng dẫn tải

1. Truy cập link Releases ở trên.
2. Chọn phiên bản mới nhất.
3. Tải file `Tool.DSP.PC_XX.zip` trong mục **Assets**.
4. Giải nén và chạy `NVMv4.exe`.

---

## 🧭 Hướng dẫn sử dụng

### 1️⃣ HOME

- Bật tăt, chỉnh sửa các Effect,..
- Thay đổi Gain các module như DAC0, DAC1, ADC0 , ADC1
---

### 2️⃣ Cài Đặt

- Đổi tên bluetooth, cài mật khẩu
- Cài đặt class âm thanh: 2.0 , 2.1, MONO, 1.1 , 3way
- Bật tắt Key và set chức năng key
- Bật tắt Biến trở VOL,..
- Bật tắt nhạc khởi động, kết nối,..
- Thay nhạc khởi động, kết nối,..
